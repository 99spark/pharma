here you can upload csv files to the data base

First click on choose file then navigate to the csv file you would like to upload

Click on the file you want to upload then click Open

The file name will now be visible next to choose file button

Now click on Upload file button

If the file has already been uploaded an alert will notify you file already exist

If file is not a csv file an alert will notify you only csv file allowed

If file passes requirments and uploads an alert will notify you file upload success
